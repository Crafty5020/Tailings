## What is Tailings

Tailings is a python code base for well a costumes tail supporting different kinds of personalities. But you may ask, how do I make the tail for my favorite chacracter? Well you need to find a TCPack maded from someone for you're character. OR just make youre own TCPack you dummy. (Im kidding with the dummy part)

### What are TCPacks

TCPacks are json files determining the tails personality button layout and more. TCPack format are a seperate ecosystem. Also TCPack aren't tied to any specific intellectual property. TCPacks in the community repository are maintained by contributors and may vary in quality. Finally are the parser is in this [github project]().
