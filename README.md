## What is Tailings

Tailings is a Python codebase for a costume tail system supporting different personalities. But you may ask, how do I make the tail for my favorite character? You need to find a TCPack made for that character, or create your own TCPack you dummy. (Im kidding with the dummy part — it’s a joke.)

### What are TCPacks

TCPacks are JSON files defining tail behavior, personality, button layout, and other configuration settings. TCPack format is a separate ecosystem. Also TCPacks are not tied to any specific intellectual property or character ownership. TCPacks in the community repository are maintained by contributors and may vary in quality. Finally the TCPack parser is available in a separate GitHub repository (coming soon).
